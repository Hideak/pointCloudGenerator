Bem-vindo ao Gerador de Nuvem de Pontos!
Versão v0.7.105 pré-alpha.

Funcionalidade Básica
--------------------------

* Para iniciar, importe um arquivo .IGES através do menu ARQUIVO > Importar Arquivo .IGES.

* Para discretizar um modelo, utilize o painel lateral GERAR NUVEM.

* Para aplicar erros de translação, utilize o painel lateral GERAR ERROS.

* Para exportar a nuvem de pontos gerada em um arquivo .pcd (Point Cloud Data) ou em um arquivo de texto plano (.txt), utilize o menu EXPORTAR.

* Para exibir as entidades presentes em um arquivo aberto, utilize o menu ENTIDADES.

Nota Importante
--------------------

O Gerador de Nuvem de Pontos funciona somente para modelos CAD representados através da entidade 186 da especificação IGES (Manifold Solid). Arquivos IGES especificados através de outra forma não serão lidos pelo aplicativo.
